在线点餐系统
=============================================================
作者：李清林
=============================================================
项目描述：
这是一个类似饿了么的外卖预定平台。
=============================================================
Tomcat 为版本9

如何运行：在导入项目后就在build path中导入mysql-connector-java-5.1.46-bin.jar

然后在不报错之后直接运行

请从  public/enter.jsp  开始本系统以获得完整过程

==============================================================

1.用户：进入主页浏览商品，查看商品详情
        直接购买商品或将商品加入购物车
        可直接下单或从购物车下单
        用户可查看自己的所有订单信息
        用户可以修改个人信息
        申请成为店家，拥有店铺
        
2.店家：可查看或管理商品（添加、修改、删除、下架）
        查看并处理用户订单
       
3.管理员：审批普通用户注册
	  审批普通用户成为商家和商家开店的申请
	  对商家将要上架的商品进行审核	
          管理店铺，可强制关闭店铺，可拒绝用户开店
          可让关闭的店铺重新开店

项目亮点：
1.支持图片上传：商品可用图片展示，可添加商品图片或修改商品图片

2.搜索与数据展示：支持按店铺或商品搜索，用到模糊查询

3.分页： 商品展示，订单展示都有进行分页
         
4.购物车： 将商品加入购物车，可查看购物车，可从购物车下单

5.导航栏： 界面清晰整洁，各个界面有导航栏方便操作，设计跟人性化

6.权限设置：界面都设置了权限判断，拥有权限才能进入相关页面
